RAGAS (Retrieval Augmented Generation Assessment) is a framework for evaluating RAG pipelines. Key metrics include: Faithfulness — measures whether the answer contains only information from the retrieved context (score 0.0 to 1.0; higher means less hallucination). Answer Relevancy — measures how relevant the answer is to the question asked. Context Precision — measures the proportion of retrieved chunks that are actually relevant to the question. In the capstone project, the eval_node performs real-time faithfulness checking with a threshold of 0.7. If the score falls below 0.7, the answer is regenerated (up to a maximum retry count). For the submission, students must prepare 5 question-answer pairs with ground truth and report baseline RAGAS scores. If RAGAS library is unavailable, manual LLM-based scoring serves as the fallback evaluation method.
