## Agentic AI Overview

Agentic AI represents a paradigm shift in artificial intelligence, moving beyond reactive systems to create autonomous entities capable of pursuing defined objectives. Traditional AI, exemplified by large language models (LLMs) responding to individual prompts, operates on a request-response basis. Agentic AI, conversely, embodies a continuous cycle of perception, decision-making, action, and learning. This is achieved by integrating LLMs with additional components to facilitate complex, multi-step reasoning and execution.

The core characteristics defining agentic systems are: **autonomy**, the ability to operate without constant human direction; **reactivity**, responding dynamically to changes within its environment; **proactivity**, taking the initiative to achieve goals rather than solely reacting to stimuli; and **social ability**, the capacity to interact with other agents (AI or human) to coordinate actions or exchange information. 

Technically, an agentic system comprises several key elements. The **reasoning engine**, typically an LLM (e.g., GPT-4, Gemini), provides the core cognitive ability – interpreting information, formulating plans, and generating actions. **Tools** extend the agent’s capabilities beyond the LLM’s inherent knowledge, enabling interaction with external APIs, databases, or other systems (e.g., a search tool, a calculator, a code interpreter). **Memory** allows the agent to retain information about past interactions and observations, crucial for learning and adapting its behavior. This memory can be short-term (context window) or long-term (vector databases). Finally, a **planning module** orchestrates the execution of complex tasks by breaking them down into manageable steps.

Modern agentic AI architectures often employ a “ReAct” (Reason + Act) paradigm, where the agent alternates between reasoning about the current state and taking actions to modify the environment. This iterative process allows for dynamic adaptation and error correction. 

This course focuses on building robust, production-ready agentic systems using **LangGraph**. LangGraph is a powerful framework designed to address the complexities of creating stateful AI workflows. It provides abstractions for managing agent state, defining tool usage, implementing memory, and incorporating human-in-the-loop feedback.  LangGraph’s built-in persistence mechanisms ensure that agent state is preserved across sessions, enabling long-running tasks and continuous learning.  Furthermore, its modular design facilitates the creation of complex agents composed of multiple sub-agents, each specializing in a specific task.
